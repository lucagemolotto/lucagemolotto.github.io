The autonomous surface vehicle (ASV) in development under the Brigantine project aims to ease the sampling of chemico-physical parameters in coastal areas. Currently we're using a multiparameter CTD for some of these parameters and a dual-camera system for capturing spectral images. I'm personally developing the software for data gathering and visualization while also contributing to various engineering aspects.
