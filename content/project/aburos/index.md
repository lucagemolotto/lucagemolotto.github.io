AbU-ROS is a novel runtime system for choreographic robot management. Built on top of ROS2 and AbU, the Attribute-based Updates DSL, it uses ECA rules for the development of choreographic routes to be used by groups of nodes. The system aims to make the coordination of swarms as easy and transparent as possible.
